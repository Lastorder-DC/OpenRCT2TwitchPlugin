# OpenRCT2 Twitch Plugin
OpenRCT2 Twitch Plugin Server & OpenRCT2 Twitch Plugin

# Requirements
- Python 3.6 or above
- gevent
- requests
- websocket-client<=0.48
- redis
- redislite

# How to install
$ pip3 install -r requirements.txt

# How to run
## Development server
$ python3 openrct2_twitch_server.py -H 0.0.0.0 -p 8000
